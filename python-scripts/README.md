# Python Scripts - Original Math Tools

Original Python implementations of the math tools before web migration.

## Files

- `exe.py` - Executable script
- `script.py` - Math play script
- `gui_app.py` - GUI application
- `script-gem.py` - Gemini integration
- `script-nblm.py` - NBLM integration
- `script-studio.py` - Studio integration

## Note

These are the original Python scripts. The web application (`mathf3-o/`) is the current active version with React + Groq AI integration.
